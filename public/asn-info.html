GeoLite2-ASN 詳細

ASN（自律システム番号）とは
ASN（Autonomous System Number）はインターネット上のネットワーク群を一意に識別する番号です。 各 ISP・企業・大学・データセンターなどが独自のASを持ち、BGPプロトコルで相互接続しています。 GeoLite2-ASN データベースはIPアドレスからそのIPが属するASNと組織名を特定します。
フィールド名 型 説明 例
asn string AS番号（"AS" プレフィックス付き） AS15169
asn_number number AS番号（数値のみ） 15169
organization string そのASを管理する組織名・ISP名 GOOGLE
